# mPasażer

Aplikacja mobilna i webowa dla pasażerów lotniczych (mPasażer) wspierająca zarządzanie lotami, odprawami i informacjami o podróży.

## Struktura projektu

Projekt składa się z następujących katalogów:
- `backend/` – Część serwerowa aplikacji (Node.js/Express)
- `frontend/` – Część kliencka aplikacji (aplikacja webowa)
- `docs/` – Dokumentacja techniczna i projektowa
